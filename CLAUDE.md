# KakaoCloud OpenAPI MCP

카카오클라우드 OpenAPI를 AI 코딩 어시스턴트에서 쉽게 활용할 수 있도록 하는 MCP 서버 프로젝트.

## 하네스: 카카오클라우드 MCP 개발

**목표:** 카카오클라우드 OpenAPI 문서를 수집/구조화하고, AI가 API 정보를 조회할 수 있는 MCP 서버를 개발한다.

**트리거:** 카카오클라우드 MCP 개발, API 수집, MCP 서버 구현, 서비스 추가, 테스트 등 MCP 개발 관련 작업 요청 시 `kakaocloud-mcp-orchestrator` 스킬을 사용하라. 단순 질문은 직접 응답 가능.
